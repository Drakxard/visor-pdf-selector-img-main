🗂️ Acceso a Carpeta de Notas

Para guardar automáticamente tus notas, necesito acceso a una carpeta en tu computadora.

Las notas se guardarán como archivos JSON y se cargarán automáticamente cuando abras el mismo PDF.

⚠️ Solo funciona en navegadores modernos (Chrome, Edge, etc.)

📝 Visor PDF Híbrido

🎮 Navegación híbrida:
• ← → Saltar páginas (estilo Chrome)
• ↑ ↓ Scroll continuo vertical
• F Pantalla completa

📝 Notas:
• N para crear notas en la página
• Haz clic donde quieras pegar la nota
• Arrastra 📝 para mover dentro de la página
• Doble clic en 📝 para editar
• Ctrl+L para LaTeX en notas
• Arrastra hacia arriba para eliminar

💾 Guardado Automático:
• Las notas se guardan automáticamente en archivos
• Se cargan automáticamente al abrir el mismo PDF
• Usa ⚙️ para configurar la carpeta de guardado
• Funciona sin conexión a internet

🖼️ Capturas:
• Ctrl + I para elegir {Teoría, Práctica} y marcar áreas
• 1er clic: esquina inicial, 2do clic: esquina opuesta (rectángulo)
• Clic dentro de la selección para eliminarla
• Ctrl + I nuevamente para guardar todas las selecciones
• Se admite marcar en varias páginas
Las imágenes se guardan en la carpeta elegida
Modal de categoría (Teoría/Práctica)

Se te pedirá seleccionar una carpeta por categoría la primera vez.
Luego se recordará automáticamente.
